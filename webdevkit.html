Base64+ & SSAE Encoding and Decoding API


This API has been used in CiphrTxt which uses Base64+ API and also has the custom SSAE (Secure Substitution Algorithm Enhanced) Encoding/Decoding, provided as a separate JS file.

Keep in mind that SSAE is still in beta, so it may contain issues and bugs.


The elements or objects defined in the API should be modified before any use in order to avoid null reference/other errors. It is defined to 2 different radio buttons reference, which should be removed accordingly.



Base64+ & SSAE Encoding/Decoding API (JavaScript file)

This API is based on the color palette creation JS algorithm in PaletteBox. To use, analyse and remove the unnecessary object references to avoid tangling between errors.
